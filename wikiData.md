# 소개 #

  * Dokuwiki의 data 플러그인 http://www.dokuwiki.org/plugin:data 의 나린위키 포팅입니다.
  * 입력 (dataentry)와 출력 (datatable, datalist 등)을 지원합니다.
  * WikiTemplate 플러그인과 같이 사용하면 입력 데이터를 보여줄 수 있습니다.
